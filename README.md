# MinePattern v1.0.0 - Minecraft Texture Generator

## 🎨 Overview
Professional pixel-perfect texture studio for Minecraft. Desktop/mobile compatible, PWA-ready.

## 🚀 Local Run
```
cd MinePattern-v1.0.0
streamlit run app_pro.py
```

## ☁️ Streamlit Community Cloud Deploy
1. Push to GitHub repo
2. Connect at share.streamlit.io
3. Main file: `app_pro.py`

## 📱 Mobile PWA/APK
1. Run app_pro.py
2. Open `index.html` in mobile browser
3. \"Add to home screen\" → Instant APK

## Features
- Resolutions: 16/32/64/128px
- Patterns: Basic, Ore, Bricks
- Custom palettes + noise/bevels
